# Removing Patches

If you need to remove the HLPatcher modifications and restore your original game installations, you have three options.

## Option 1: Restore from Backup (Recommended)

If you chose to create a backup during patching, you can easily restore any game to its original state:

1. Navigate to your **Documents** folder.
2. Look for a folder named after the game followed by the backup date (e.g. `Half-Life backup (YYYY-MM-DD)`).
3. Delete the current patched game folder from your Steam library.
4. Copy the backup folder back to its original location.
5. Rename it back to the original folder name if needed (e.g. `Half-Life`, `Portal`, `Half-Life 2`).

## Option 2: Verify Game Files through Steam

If you didn't create a backup or prefer using Steam's built-in verification:

1. Open **Steam** and go to your **Library**.
2. Right-click the game you want to restore.
3. Select **Properties** from the context menu.
4. Go to the **Installed Files** tab.
5. Click **Verify integrity of game files**.
6. Steam will automatically download and restore any modified or missing files.

!!! note
    Each game must be verified separately. Verifying one game will not affect others in your library.

## Option 3: Complete Removal

For a completely clean slate:

1. Uninstall the game through **Steam**.
2. Manually delete the game's installation folder from your Steam library to remove any leftover files.
3. Reinstall the game through **Steam**.

This ensures all patched files and modifications are completely removed from your system.
